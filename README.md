# ionicluster

A Python package for high-throughput structural and cluster analysis in molecular simulations. `ionicluster` also provides specialized utilities for the post-processing of [infRETIS](https://github.com/infretis/infretis) (Replica Exchange Transition Interface Sampling) simulation paths.

## Features

* **Structural Analysis:**
    * Radius of Gyration ($R_g$)
    * Coordination Numbers (CN)
    * Radial Distribution Function (RDF)* Note that this implementation specifically calculates the RDF **within** identified clusters, rather than bulk systems; see the supporting literature for the modified mathematical framework.
    * Xth-minimum distance
* **Topological Cluster Analysis:** Graph-based algorithms utilizing `scipy.sparse.csgraph` to identify, quantify, and analyze the distribution of simulation entities (atoms, ions).
* **InfRETIS Post-processing:** Dedicated workflows to work with the pathways generated by InfRETIS simulations.
* **Ecosystem Integration:** Built on top of `MDAnalysis` to handle simulation trajectories generated with different software; `NumPy` and `SciPy` enable high-performance numerical and graph-based operations.

This package was used to obtain analysis presented in the paper `in development`. This paper also contains the detailed description of the math of the listed structural parameters.

## Usage & Examples

For a quick start, please refer to the scripts provided in the `paper_examples/` directory. These scripts demonstrate how to initialize the `ClusterAnalyser` and `StructuralAnalyser` classes on real simulation data.

Moreover, this directory contains the **necessary input files and atomic configurations** required to reproduce the results presented in our paper.
